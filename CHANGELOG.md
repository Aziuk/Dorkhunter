## [1.0.0] — 2024-06-05 — Initial Public Release

### Added
- 50+ preloaded dorks across 9 categories
- **Dork Builder** — visual guided form with 8 operator fields and live query preview
- **Operator Cheatsheet** — inline reference panel with all operators, examples, and tips
- **Pin to top** — pin any dork to always appear first in the list
- **Multi-engine support** — Google, Bing, and DuckDuckGo with per-engine compatibility notes
- **Copy query** — one-click copy of raw dork query to clipboard
- **Target memory** — last 5 used targets saved as quick-select chips
- **Export / Import** — full dork library backup and restore as JSON
- **Engine badges on cards** — each dork shows which search engines support it
- **Keyboard shortcuts** — `A` add dork, `B` builder, `/` search, `Esc` close panel
- **Categories & Tags** — fully customizable, filter by either
- **Cloud & APIs** category — S3, Azure, GCP, GitHub, Pastebin dorks
- **CMS & Frameworks** category — WordPress, Drupal, Laravel, Django, Jenkins dorks
- **Full CRUD** — add, edit, delete, pin your own dorks
- 100% local — all data stored in browser, zero external servers
